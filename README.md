# 🌐 Pratham Shinde — Personal Portfolio

> **AI & Data Science Student · Graphic Designer · Chembur, Mumbai**

A sleek, dark-themed personal portfolio website built with pure HTML & CSS — no frameworks, no dependencies. Features a custom animated cursor, scroll-based fade-in animations, and a fully responsive layout.

---

## ✨ Features

- 🎨 **Custom animated cursor** with lag-ring effect
- 🌙 **Dark cyberpunk aesthetic** with teal accent (`#00e5c3`) and grid background
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- ⚡ **Zero dependencies** — plain HTML, CSS, and vanilla JS
- 🖼️ **Profile photo box** with gradient overlay
- 🔽 **Scroll fade-up animations** using Intersection Observer API
- 🔗 **Direct links** to Email, LinkedIn, GitHub, and Instagram
- 📄 **CV download** button

## 🚀 Getting Started

### View Locally
Just open `index.html` in any modern browser — no build step, no server needed.

```bash
# Clone the repo
git clone https://github.com/your-username/portfolio.git

# Open in browser
open index.html
```

### Deploy on GitHub Pages
1. Push the repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://your-username.github.io/portfolio/`

## 📸 Adding / Updating Your Photo

1. Place your photo in the project root folder
2. In `index.html`, find `<img src="pic.jpeg" ...>`
3. Replace `pic.jpeg` with your filename

---

## 🎨 Customization

All design tokens are CSS variables at the top of `index.html`:

```css
:root {
  --bg: #080c10;        /* Main background */
  --accent: #00e5c3;    /* Teal highlight color */
  --text: #c8d8e8;      /* Body text */
  --text-dim: #6a8090;  /* Muted text */
}
```
